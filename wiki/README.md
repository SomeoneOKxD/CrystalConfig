# CrystalConfig Wiki

This directory contains the GitHub Pages source for the developer-facing CrystalConfig wiki. The root workflow at `.github/workflows/wiki-pages.yml` publishes this directory with Jekyll.

The wiki is for developers who want to use CrystalConfig in their own Fabric mods. It should document the official `SomeoneOKxD/CrystalConfig` JitPack dependency, runtime metadata, API usage, widgets, persistence, and screen-opening patterns.
